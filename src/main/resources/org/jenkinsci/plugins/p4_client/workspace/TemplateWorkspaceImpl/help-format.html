Workspace name formatter

Jenklin slave nodes must each use a unique Perforce workspace. The format string configures the workspace name by substituting the following tokens: (at least one token must be used)

    ${node}
    The name given to the slave Jenkins node. (Caution this might not be unique).
    ${hostname}
    The hostname for the slave Jenkins node.
    ${project}
    The name of the Jenkins build Job. (Caution this is not unique for slaves).
    ${hash}
    Unique hash code of the Jenkins node.
